## Usage
### Install
```bash
git clone https://github.com/beyzaerkan/error-management.git
cd error-management
```

### Setup
Install dependencies
```bash
npm install
```

### Set Environment Variables
```bash
cp sample.env .env
```

### Start
```bash
npm start
```
